snap find

manpage

執行

$ man snap

可以找到一段說明

COMMANDS

...略...

   find
       Finds packages to install

       The find command queries the store for available packages.

help

執行

$ snap -h

可以找到一段說明

Usage:
 snap [OPTIONS] <command>

Help Options:
 -h, --help  Show this help message

Available commands:

...略...

 find        Finds packages to install

執行

$ snap find -h

或是執行

$ snap find --help

則會顯示

Usage:
  snap [OPTIONS] find [<query>]

The find command queries the store for available packages.

Help Options:
  -h, --help         Show this help message
